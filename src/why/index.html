What would I like to get out of this course?

18 months ago I made the decision I wanted to go back into software development, as I realised:

  1. Having tried management, it I find it it does not motivate me
  2. I enjoy the challenge of writing software
  3. I like the satisfaction I get when I have built something and it works !

Being largely self taught I know there is alot of room for improvement in the way I write code, and so I would like to gain the following:

  • Software design techniques - Wireframes, design principals etc
  • Coaching from experienced developers
  • A better understanding of the "right way" to do things
  • Angular
  • Java
  • 100% comfortable with CSS

In short this course presents a wonderful opportunity to develop my existing skills to the next level and beyond. I want to be a better software developer, confident in my skills and working back in software development.
